Postman API自动化05--上传文件

Posted on 2019-06-11 | In API自动化

目标

能通过Postman工具实现接口上传文件；

获取关键参数

我们先直接在页面操作文件上传功能，在网络中查看请求详情；
【Form Data】部分有我们需要的关键参数：name=”file”

配置脚本

1、拿到上面的关键参数之后，我们在请求的Body中选择form-data模式；
（有没有发现与上面【Form Data中看到的Content-Disposition: form-data;有点关系）
2、切换Key的类型为File，在Value中选择要上传的文件即可；

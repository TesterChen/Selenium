关于JMeter

官网关于JMeter是这样描述的：
The Apache JMeter™ application is open source software, a 100% pure Java application designed to load test functional behavior and measure performance. It was originally designed for testing Web Applications but has since expanded to other test functions.

JMeter是一款用Java写的开源软件，原来主要是用于WEB性能测试，后来扩展了很多其他类型的测试。

官网有详细的介绍和使用教程，有兴趣深入了解的同学们可以详细阅读官网的教程。
官网全英文教程链接在这里：http://jmeter.apache.org/usermanual/index.html

JMeter是一款无需安装的软件，下载Binaries版之后直接运行下载包中/bin/ApacheJMeter.jar即可启动软件；
“JMeter 4.0 is compatible with Java 8 or Java 9”，JMeter 4.0以上版本需要Java 8以上的版本；
有更多自定义需求的可以下载Source版进行修改；
下载地址：http://jmeter.apache.org/download_jmeter.cgi

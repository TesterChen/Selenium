LIMIT

LIMIT是MYSQL的语法。
SELECT * FROM TABLE LIMIT M,N
其中M是指记录开始的index，从0开始，表示第一条记录，N是指从第M+1条开始取N条。
M可以被省略，也就是只传入N，表示从0开始取N条记录。

SELECT * FROM TABLE LIMIT 2,4
即取出第3条至第6条，4条记录。

SELECT * FROM TABLE LIMIT 4
即取出第1条至第4条，4条记录。

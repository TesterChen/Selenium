Postman API自动化03--环境变量与全局变量

Posted on 2019-05-19 | In API自动化

环境信息

目标

了解环境变量与全局变量的差异及使用场景；

直观感受

我们先通过两张图片直观地感受环境变量与全局变量；

通过上面的两张图，我们看到切换环境变量时，全局变量并不会随着变化，而是一直都显示，说明全局变量一直可以被调用；

环境变量：顾名思义与环境有关，如：不同环境执行测试脚本时配置的用户名、密码应该被放在环境变量中存储；
全局变量：不管我们在哪个环境下执行脚本，他的值是不需要调整的，或者调整之后，其他环境执行时也是可以的，如：登录成功的文字标识；

作用范围

1、环境变量与全局变量的作用范围为Wrokspace，不同Wrokspace之间的变量相互隔离，互不影响；
2、当环境变Key与全局变量Key重复时，优先使用环境变量中的值，即使环境变量中对应Key的值为空；

赋值与使用

环境变量与全局变量可以被修改；

使用方式均是直接通过{{var}}形式；
